# AsheOs

AsheOs é um projeto demonstração de sistema fullstack de cadastro de Ocorrências, desenvolvido com as seguintes tecnologias:

- Frontend: React.js (Next.js)
- Backend: Java 17 (Spring Boot)
- Banco de Dados: PostgreSQL
- Containerização: Docker

## Definição

Sistema básico de registro e gerenciamento de usuários, endereços, ocorrências e armazenamento de arquivos anexados.
